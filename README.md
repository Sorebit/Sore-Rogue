# Sore-Rogue
A Brogue inspired but much simpler rouge-like

## Preview
![preview](/assets/preview.png)

## Usage
`make` creates an executable `Rogue`

Dependencies:
- `ncurses`

You will need a 256-color terminal in order to run this.
